# アプリケーション名
Food Loss
# アプリケーション概要
廃棄する食べ物、飲み物、訳あり商品、を販売するアプリです  
＜作ろうと思ったキッカケ＞  
コロナ禍の影響で飲食店の営業時間短縮により、第一次産業(農家、漁師)にまで影響があることをテレビで知り  
このアプリを作ろうとを思いました。また　恵方巻等の廃棄量の話を聞いた覚えがあるので、廃棄量を減らす  
目的もあり、SDGsも意識しました。

・ログイン機能
